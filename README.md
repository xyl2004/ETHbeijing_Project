# ETHbeijing黑客松管理平台

基于区块链技术的黑客松管理平台实现方案。项目采用前后端分离架构，结合智能合约实现去中心化队伍管理、NFT奖励发放和自动化流程控制，构建完整的Web3活动管理解决方案。

## 主要功能

- 去中心化身份管理模块
- 自动化流程控制模块
- 混合架构数据同步

详细技术实现请参考[区块链黑客松管理平台技术实现与运行指南](区块链黑客松管理平台技术实现与运行指南.md)。
